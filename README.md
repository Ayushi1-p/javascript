
# 📘 JavaScript Data Types Explorer

A simple educational project that explains JavaScript primitive data types vs objects with practical examples.

---

## 🚀 Features

- 📖 Clear explanation of primitive data types
- 🧠 Understanding objects in JavaScript
- ⚡ Interactive examples using JavaScript
- 🎯 Beginner-friendly learning approach

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
